# Retail Sales Performance Dashboard

## Project Overview

This project is an end-to-end **Retail Sales Analytics Dashboard** built using **Snowflake and Power BI**. The objective of the dashboard is to analyze retail sales data and generate actionable insights to support data-driven decision making.

Sales data is stored and processed in **Snowflake**, queried using **SQL**, and then connected to **Power BI** for visualization. Key performance indicators are created using **DAX measures** to evaluate sales performance, profitability, and trends.

---

## Business Objective

The goal of this project is to build a dashboard that helps stakeholders:

* Monitor overall **sales performance**
* Analyze **profit and profit margins**
* Identify **top-performing regions and products**
* Track **sales trends over time**
* Support **strategic business decisions**

---

## Tools & Technologies

* **Power BI** – Data visualization and dashboard creation
* **Snowflake** – Cloud data warehouse
* **SQL** – Data querying and transformation
* **DAX** – Analytical calculations and KPI measures

---

## Key Metrics & KPIs

The dashboard includes the following key metrics:

* Total Sales
* Total Profit
* Profit Margin %
* Sales YTD (Year-to-Date)
* Region-wise Sales Analysis
* Product Performance Insights

These KPIs help analyze both **financial performance and operational efficiency**.

---

## Dashboard Features

* Interactive KPI cards for key business metrics
* Region-wise and product-wise sales analysis
* Time intelligence analysis using DAX
* Interactive slicers and filters
* Drill-down capability for deeper insights
* Conditional formatting for better visualization
