# Retail Sales Performance Dashboard

## Overview
This project is an end-to-end analytics solution integrating Power BI with Snowflake to analyze retail sales data.

## Tools Used
- Power BI
- Snowflake
- SQL
- DAX

## Key Features
- Total Sales, Total Profit, Profit Margin %, Sales YTD
- Interactive slicers and drill-down
- Conditional formatting
- Time intelligence analysis

## Business Insights
- Identifies top-performing regions and products
- Tracks sales trends over time
- Supports data-driven decision making
